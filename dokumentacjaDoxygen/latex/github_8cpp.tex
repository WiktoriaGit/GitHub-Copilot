Funkcja główna programu. 

Testuje różne funkcje i operacje na klasie matrix, takie jak tworzenie macierzy, wstawianie wartości, operacje matematyczne i inne.

\begin{DoxyReturn}{Returns}
int Zwraca 0, jeśli program zakończył się poprawnie. 
\end{DoxyReturn}
Test 1\+: Tworzenie i wyświetlenie macierzy.

Tworzy macierz o zadanym rozmiarze, losowo wypełnia ją elementami i wyświetla wynik.

Test 2\+: Wstawianie i pobieranie wartości.

Wstawia wartości do określonych pozycji w macierzy, wyświetla wynik, a następnie pobiera te wartości i wyświetla je.

Test 3\+: Odwracanie macierzy.

Wyświetla macierz przed i po odwróceniu wierszy i kolumn.

Test 4\+: Wypełnianie przekątnej.

Wypełnia przekątną macierzy zadanymi wartościami i wyświetla wynik.

Test 5\+: Wypełnianie kolumny i wiersza.

Wypełnia określoną kolumnę i wiersz zadanymi wartościami, a następnie wyświetla wynik.

Test 6\+: Wyświetlanie przekątnej, nad i pod przekątną.

Wyświetla przekątną, elementy nad przekątną i pod przekątną macierzy.

Test 7\+: Operacje arytmetyczne na macierzach.

Wykonuje operacje dodawania i mnożenia macierzy oraz operacje dodawania i mnożenia macierzy przez skalary.

Test 8\+: Operatory porównania.

Sprawdza równość, większość i mniejszość między macierzami.

Test 9\+: Operatory inkrementacji i dekrementacji.

Testuje operator zwiększania (++) i zmniejszania (--) na macierzy.

Test 10\+: Szachownica.

Wyświetla macierz w formie szachownicy, gdzie pola białe mają wartość 0.
